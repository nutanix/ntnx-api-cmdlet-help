# Get-StorageContainer

## SYNOPSIS
Displays storage containers.

## SYNTAX

```
Get-StorageContainer [-Page <Int32>] [-Count <Int32>]
 [-Servers <System.Collections.Generic.HashSet`1[System.String]>] [<CommonParameters>]
```

## DESCRIPTION
Displays storage containers available for the connected Prism Central. Typically used to get UUID of the storage container to attach to a VM.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-StorageContainer
```

Displays a list of available storage containers for the connected Prism Central.
